# IntelliBI Frontend

React + TypeScript frontend for the IntelliBI Business Intelligence Platform.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

The app will be available at http://localhost:3000

## Build

```bash
npm run build
```

## Environment Variables

Create a `.env` file:

```
VITE_API_BASE_URL=http://localhost:8000/api/v1
```
